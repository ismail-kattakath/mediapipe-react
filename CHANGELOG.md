# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-11

### Added

- **LLM Streaming Hook**: `useLlm` for real-time generative AI inference in the browser.
- **SSR-safe Provider**: `MediaPipeProvider` with automatic hydration safety and context management.
- **Multi-framework support**: Optimized for Next.js App Router and vanilla React.
- **Subpath exports**: Direct entry points for `@ismail-kattakath/mediapipe-react/genai` and `vision`.

### Planned (Roadmap)

- **Vision Tasks**: Hooks for Hand Landmarker, Object Detection, and Face Mesh.
- **Audio Tasks**: Audio classification and speech-to-intent.
- **Custom Models**: Capability to load generic TFLite models.
